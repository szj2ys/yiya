# Yiya — Mission & Vision

## Mission

**Make daily language learning feel effortless.**

We believe language ability is built through consistent small habits, not marathon study sessions. Every feature we ship should make the next 5-minute lesson easier to start, more rewarding to finish, and harder to skip.

## Vision

**Help 100 million people build a second language through 5 minutes a day.**

## Core Beliefs

1. **Consistency beats intensity.** A 5-minute daily streak outperforms a 2-hour weekend cram. We design for habit formation first.
2. **Science over intuition.** Spaced repetition (FSRS), AI-powered feedback, and adaptive difficulty are not nice-to-haves — they are the foundation.
3. **Motivation is a product feature.** Streaks, XP, hearts, quests, and leaderboards are not gimmicks. They are the scaffolding that keeps learners showing up.
4. **The goal is real conversation.** Vocabulary drills are the means, not the end. Every feature should move the learner closer to speaking with confidence.
5. **Great education should be accessible.** Free to start, no credit card required. AI-generated content lowers production cost so we can serve more languages and more learners.

## Decision Filter

When evaluating any new feature, bug fix, or design change, ask:

> **"Does this make daily learning feel more effortless?"**

- If yes → prioritize it.
- If no → question whether it belongs in the product.

## Strategic Pillars (2026)

| Pillar | What it means | Key initiatives |
|---|---|---|
| **Habit Engine** | Make it frictionless to learn every day | Streak mechanics, daily quests, push reminders, social sharing |
| **Smart Learning** | Personalize the learning path with science + AI | FSRS review, AI explanations, adaptive placement test, AI variant questions |
| **Real Output** | Move learners from recognition to production | TYPE challenges, AI conversation practice, pronunciation feedback |
| **Content Scale** | More languages, more topics, faster | AI content generation pipeline, community contributions |

## Product North Star Metric

**Daily Active Learners (DAL)** — the number of unique users who complete at least one lesson per day.

This single metric aligns with the mission: if daily learning feels effortless, more people will do it every day.

## Tagline

> Speak confidently. One lesson at a time.
